Default Button

To use, place the following class .buton

Example:

Code snippet:

						<button class="button">default</button>

						<button class="waves-effect waves-light button">default</default with ripple>

Disabled Button

To use, place the following class .buton-disabled or add attribute disbaled in the button

Example:

Code snippet:

						<button class="button-disabled">disabled</button>

						<button class="button" disabled>disabled</button>

Large Button

To use, place the following class .buton-disabled or add attribute disbaled in the button

Example:

Code snippet:

						<button class="waves-effect waves-light button-large">large</button>

						<button class="waves-effect waves-light button-large-disabled">button-large-disabled</button>
