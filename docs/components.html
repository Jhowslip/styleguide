Button

These are notes associated with this button component.

Default Button

To use, place the following class .button and will be dispalyed the default button without effects. This button has a larger height for buttons that need more attention.

If you want a ripple effect, you can add waves-effect and waves-light and will appear on click action.

Example:

Code snippet:

				<button class="button">default</button>

				<button class="waves-effect waves-light button">default with ripple</button>

Disabled Button

To use, place the following class .buton-disabled or add attribute disbaled in the button.

Example:

Code snippet:

				<button class="button-disabled">disabled</button>

				<button class="button" disabled>disabled</button>

Small Button

To use, place the following class .buton-disabled or add attribute disbaled in the button

Example:

Code snippet:

				<button class="waves-effect waves-light button-small">small</button>

				<button class="waves-effect waves-light button-small-disabled">button-small-disabled</button>

Secondary Button

To use, place the following class .buton-disabled or add attribute disbaled in the button

Example:

Code snippet:

				<button class="waves-effect waves-light secondary-button">large</button>

				<button class="waves-effect waves-light button-large-disabled">button-large-disabled</button>

Button with Icon

Example:

Code snippet:

				<button class="button waves-effect waves-light">Default button with icon</button>

				<button class="secondary-button-disabled waves-effect waves-light">Disabled button</button>

				<button class="secondary-button waves-effect waves-light">Secondary Button</button>

Button Tiny

Example:

Code snippet:

				<button class="button-tiny waves-effect waves-light">Save</button>

These are notes associated with this button component.

Inputs

You must have a .input-field div wrapping your input and label. This helps our JS animate the label. This is only used in our Input and Textarea form elements.

Input Disabled

Example:

Code snippet:

				<div>

				 <input id="full_name" type="text" disabled>

				 <label for="full_name">Full Name</label>

				</div>

Input With Validation

The validate class leverages HTML5 validation and will add a valid and invalid class accordingly. If you don't want the Green and Red validation states, just remove the validate class from your inputs.
