Select allows user input through specified options. Make sure you wrap it in a .input-field for proper alignment with other text fields. Remember that this is a jQuery plugin so make sure you initialize this in your document ready.

Example:

Code snippet:

      <div class="input-field">

       <select>

        <option value="" data-graph="icon-interrogation" disabled selected>Choose your option</option>

        <option value="" data-graph="icon-delivery-truck">option 1</option>

        <option value="" data-graph="icon-acessocard">option 2</option>

        <option value="" data-graph="icon-acessocard">option 3</option>

       </select>

      </div>
