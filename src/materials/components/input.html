--- notes: | These are notes associated with this `button` component. ---

Inputs

You must have a .input-field div wrapping your input and label. This helps our JS animate the label. This is only used in our Input and Textarea form elements.

Input Disabled

Example:

Code snippet:

				<div>

				 <input id="full_name" type="text" disabled>

				 <label for="full_name">Full Name</label>

				</div>

Input With Validation

The validate class leverages HTML5 validation and will add a valid and invalid class accordingly. If you don't want the Green and Red validation states, just remove the validate class from your inputs.

Example:

Valid name ;P
Wrong name format
Name cannot contain number
Recusandae voluptatibus consequuntur eius. Fugit saepe ex, porro laudantium molestiae laborum dolore vel doloribus, pariatur, quis eligendi. Excepturi itaque quo, dolorum tempore?

Code snippet:

				<div>

				 <input id="full_name" type="text" class="validate">

				 <label for="full_name">Full Name</label>

				</div>
