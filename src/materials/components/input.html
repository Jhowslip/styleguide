--- notes: | These are notes associated with this `button` component. ---

Inputs

You must have a .input-field div wrapping your input and label. This helps our JS animate the label. This is only used in our Input and Textarea form elements.

Input Disabled

Example:

Code snippet:

				<div>

				 <input id="full_name" type="text" disabled>

				 <label for="full_name">Full Name</label>

				</div>
