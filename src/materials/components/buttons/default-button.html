To use, place the following class .button and will be dispalyed the default button without effects. This button has a larger height for buttons that need more attention.

If you want a ripple effect, you can add waves-effect and waves-light and will appear on click action.

Example:



Code snippet:

				<button class="button">default</button>

				<button class="waves-effect waves-light button">default with ripple</button>
